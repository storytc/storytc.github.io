Git和Github托管代码

Author: Ariel Date: 七月 30, 2023  12:02:45 Category: Bioinformatics

1. 提交本地代码

这一步添加要上传的文件/目录到暂存区

  • git add filename

这一步给暂存区生成快照并提交

  • git commit -m ‘filename/description’

提交至远程仓库

  • git push origin master #具体分支根据需求更改

2. 其他命令

查看状态

  • git status

查看历史提交信息

  • git log

分支的用法

  • git branch #查看分支

  • git branch -a #查看所有分支

  • git branch -l #查看本地分支

  • git branch name #创建本地name分支

  • git checkout name #切换到name分支

  • git checkout -b name #创建一个新的本地分支并切换到新分支

  • git push origin name #将新本地分支推送至远程

  • git branch -d name #删除本地name分支

  • git push origin :name #删除远程name分支

更新本地仓库至最新改动

  • git pull

内建的图形化git：

  • gitk

恢复到上一个版本

  • git reset –hard HEAD^

显示版本差异

  • git diff #主要是显示暂存区与缓存区的修改
