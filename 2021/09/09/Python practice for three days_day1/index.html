Why Python?

  • Python是一个开源软件，免费
  • Python是一门胶水语言，这意味着Python能让你快速上手，并且相对于其他编程语言来说，Python速度相对较快
  • Python有着庞大而活跃的社区，标准库和包都很齐全
  • Python应用广泛（Google、NASA…）

Some good habit in programming

  • 坚持使用缩进和反斜杠，不要让你的脚本乱七八糟；
  • 变量命名尽量使用描述性的名称，不要随意命名变量；
  • 对代码写批注，有利于你的同伴或者将来的自己理解代码；
  • 不要同时学习多种语言；

Reading/writing files

1.Built-in open() functions

我们可以通过内置的open()函数来打开文件，open返回一个文件对象，最常使用两个参数:

​ file_object = open(filename, mode)

mode可以是：

  • ‘r’：表示只读；
  • ‘w’：表示只写，当存在同名文件时旧文件会被重写；
  • ‘a’：打开文件进行追加，任何数据都会自动被追加到文末；
  • ‘r+’：可读可写；
