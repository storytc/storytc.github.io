LaTex_index_one

Author: Ariel Date: 十一月 28, 2021  14:59:17 Category: programming

First LaTex document

1. Basic layout

A “Hello World” beginning

The statements in LaTex are mostly begin with a backslash \, and all commands share the same structure: \commandname{option}

The first line documentclass define the layout of the document, other sort of layout such as book and et.，\begin和\end are not commands actually，they defines an environment. It is possible to contain multiple environments in a document. An environment is simply a specific typesetting in part of your document.

2. Adding a title page

Now we can see that the statements\title, \date, \author are not in our document environment. These statements create an area that before the main document is called preamble. We use these statements to set up the values of \maketitle, it will automatically build a titlepage and \newpage speaks for itself.

In order to hide the page number in titlepage, we can remove it by adding the \pagenumbering{gobble} and changing it back to the next page with \pagenumbering{arabic}

Summary

  • A document has it class and multiple environments.
  • All commands begin with backslash.
  • You can add preamble before the main document for a brief introduction and hide the page number.
  • Useful settings for pagenumbering
    • gobble – no numbers
    • arabic – arabic numbers
    • roman – roman numbers
