Insert an image in LaTex

1. Captioned images

Undoubtedly, it is feasible to insert images in LaTex! All the figures will be indexed autamatically using figure environment and graphicx package.

and the output is:

Now, let’s explain the parameters. From line 5 to line 9 are the commands to insert the figure. First of all, we define the figure environment. In order to include a figure, you must use \includegraphics command. It takes an option of image width in brakets and the path to your image. The command width=\linewidth means that I want to scale the width of image to fit the width of document. The image is in the same path of my .tex file, so I simply put the name of my image file. The \label parameters is invisible in our output , but it is useful if you want to refer it.

2. Image positioning

To order the figure in specific location, we might to use some parameters.

By adding [h!] will set the figure to be shown in the certain position. Other posible values are:

  • h(here) —- same location
  • t(top) —- top of page
  • b(bottom) —- bottom of page
  • p(page) —- on the extra page
  • !(override) —- will force the specified location

3. Multiple inmages

It is also natural to add multiple images in a document. In such situations, we need to use another environment called subfigure.

and the output is:

I sepecifid the location of pics manually with \begin{subfigure}[b]{0.4\linewidth} means that two pictures has the same width of 0.4. It is flexible to set the width for multiple images. But take the whole document width into consideration and don’t exceed it.

Summary

  • Use the graphicx package to enbed figures
  • Specify the location of images manually
  • Refer to the images by setting \label and use the \ref tag.
